What to use when

  • Online Predictions: Managed endpoint for real-time serving.
  • Batch Predictions: Async jobs reading/writing GCS/BQ.
  • Vector Search: ANN index for RAG with embeddings.
  • Cloud Run (GPU) or GKE: Self-host OSS model servers (vLLM/Triton) when you need custom runtimes.
